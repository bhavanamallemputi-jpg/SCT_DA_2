# 📊 SkillCraft Technology - Data Analytics Internship

## Task 1: Interactive Sales Dashboard using Microsoft Excel

### 📌 Project Overview
This project was completed as part of the **SkillCraft Technology Data Analytics Internship**.

The objective was to analyze the **Sample Superstore** dataset and build an interactive sales dashboard using Microsoft Excel. The dashboard provides valuable business insights through KPIs, Pivot Tables, Pivot Charts, and Slicers.

---

## 🎯 Objective

- Analyze sales performance.
- Visualize business trends.
- Identify high-performing categories and regions.
- Create an interactive dashboard for decision-making.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## 📂 Dataset

**Sample Superstore Dataset**

The dataset contains information about:
- Orders
- Customers
- Products
- Sales
- Profit
- Quantity
- Region
- Category
- Segment

---

## 📈 Dashboard Features

### KPI Cards
- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Orders
- 💵 Average Sales

### Visualizations
- Sales by Category
- Monthly Sales Trend
- Profit by Region
- Top 10 Products by Sales

### Interactive Filters
- Region
- Category
- Segment
- Year

---

## 📊 Business Insights

- Technology generated the highest sales.
- West Region recorded the highest profit.
- Sales showed steady growth throughout the year.
- Top-selling products contributed significantly to overall revenue.
- Interactive filters allow users to explore the data dynamically.
